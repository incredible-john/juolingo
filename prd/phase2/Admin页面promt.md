我需要你帮我设计一个 ADMIN 页面，用来做我的英语学习软件。这个 ADMIN 页面总体来说分为五列：

1. 第一列：用来编辑主题 SUBJECTS。它可以调整不同 SUBJECTS 的顺序，以及编辑 SUBJECTS 的主标题和副标题。
2. 第二列：用来编辑单元 UNITS。它可以编辑 UNIT 的主副标题，以及通过拖动调整 UNITS 的顺序。
3. 第三列：用来编辑课程 LESSONS。同样可以编辑 LESSON 的标题，并通过拖动调整不同 LESSON 的顺序。
4. 第四列：用来编辑问题 CHALLENGES。同样可以拖动调整顺序，并编辑 CHALLENGE 的文本。
5. 第五列：用来编辑 CHALLENGE 问题下面的选项。选项部分只需要支持修改文本即可。

页面的交互逻辑如下：
1. 第一列展示所有主题。
2. 当我选择第一列里面的某个主题时，第二列会展示该主题下面所有的单元。
3. 当我点击第二列里面的某一个单元时，第三列会展示这个单元下面的所有课程 LESSON。
4. 当我点击第三列里面的课程时，第四列会展示这个课程下面的所有 CHALLENGES 问题。
5. 第五列则会展示该问题下面的所有选项。
